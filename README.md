**브랜치 전략 - [아래 참고]/issue number**

- feat - 새 기능
- fix - 변경사항(변수, css 등)
- refactor - 구조 변경 ex) api 전후 데이터 불러오기 변경?
- Test - 테스트 코드
- chore - 환경 설정, 주석 제거, 이미지 파일 추가 등
